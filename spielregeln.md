# Spielregeln - Schach

## Figuren:
 - Bauer, Springer, Läufer, Turm, Dame, König

## Farben:
 - Weiß, Schwarz

## Spielfeld:
 - 8x8, schwarz-weiß kariert, weißes Feld rechts unten

## Startaufstellung:
  ```
  B B B B B B B B
  T S L X X L S T
  ```
Dame und König kommen auf die 2 X Felder: weiße Dame - weißes Feld, schwarze Dame - schwarzes Feld

## Figurzüge:
 - **Bauer:** Immer nur ein Feld Richtung Gegner / auf seinem eigenen Startfeld kann er 2 Felder ziehen / er kann nur eine Figur schlagen, die diagonal rechts oder links von ihm steht / Bauer kann durch eine Figur, die vor ihm steht blockiert werden / erreicht er die gegnerische Endlinie kann er zu Springer, Läufer, Turm oder Dame umgewandelt werden
 - **Springer:** Zieht in der Form des Buchstaben "L" (2 Felder in eine Richtung, eins nach rechts oder links) / kann über Figuren springen / kann in jede Richtung ziehen / schlägt genauso wie er zieht
 - **Läufer:** Zieht diagonal über das Spielbrett / muss vor Figuren anhalten oder sie schlagen, kann nicht springen / kann in jede Richtung ziehen
 - **Turm:** Zieht horizontal und vertikal über das Spielbrett / muss vor Figuren anhalten oder sie schlagen, kann nicht springen / kann in jede Richtung ziehen
 - **Dame:** Zieht diagonal, horizontal und vertikal  über das Spielbrett / muss vor Figuren anhalten oder sie schlagen, kann nicht springen / kann in jede Richtung ziehen
 - **König:** Zieht in jede Richtung, aber ein Feld pro Zug / schlägt genauso wie er zieht

## Situationen:
 - **Schach:** Wenn eine Figur den König angreift / Verteidigung: Angreifer schlagen, Figur dazwischenstellen (führt zu Fesselung), König aus der Gefahr ziehen
 - **Schachmatt:** Wenn der Gegner Schach gibt und man nicht mehr verteidigen kann
 - **Patt:** Wenn kein Zug mehr ermöglicht werden kann oder nicht genügend Figuren am Brett sind, um das Spiel zu beenden
 - **Fesselung:** Eine Figur kann nicht ziehen, da sie sonst einen Angriff auf den König freigibt

## Besondere Züge:
 - **Rochade:** Der König und seine Türme können gleichzeitig ziehen und Seiten wechseln: Felder zwischen König und Turm müssen leer sein und dürfen nicht vom Gegner angegriffen werden / König darf nicht angegriffen werden  / König und Turm dürfen davor nicht gefahren haben // König zieht 2 Felder Richtung Turm, Turm stellt sich auf das Feld auf der anderen Seite neben den König
 - **En Passant:** Ein Bauer zieht 2 Felder vorwärts und neben einem gegnerischen Bauern zum stehen / dieser kann ihn sofort mit einem diagonalen Zug schlagen / ist nur sofort erlaubt
